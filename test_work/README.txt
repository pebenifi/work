Сервис уведомлений для fabrique.

Сервис управления рассылками API администрирования и получения статистики.

-------------------------------------------------------------------------------------------------
Как установить?
1- Откройте Pycharm или другой IDE.
2- Клонируйте репозиторий:
3- Установите необходимые библеотеки из файла requirements.txt , вставив команды ниже в терминал.
        cd test_work
        pip install -r requirements.txt
4- Произведите миграции, вставив 2 команды ниже в терминал.
        cd test_work
        python manage.py makemigrations
        python manage.py migrate
5- Запустите проект, вставив команду ниже в терминал.
        python manage.py runserver
-------------------------------------------------------------------------------------------------

Документация доступна после запуска проекта по ссылке
http://127.0.0.1:8000/docs/
